# Watermark Studio

**简体中文** | [English](README.md)

一个本地优先的图片水印工具，用于批量添加 PNG 水印和文字水印。所有图片处理都在浏览器本地完成，不会上传到服务器。

在线访问：https://max-lu-4416.github.io/Watermark-Studio/

## 功能

- 批量导入 JPG、PNG、WEBP 图片。
- 快速渐进导入：前几张优先显示，后续图片并行处理。
- 支持 PNG 水印、文字水印，或两者组合。
- 支持横排、竖排、仅图片、仅文字四种水印布局。
- 可调整大小、透明度、边距、反色和九宫格位置。
- 支持深色 / 浅色模式，带切换动画和对应图标。
- 支持导出 JPG、PNG、WEBP。
- 导出设置包含文件命名、品质、色彩空间、文件大小限制、图片尺寸、元数据和输出锐化。
- 批量图片可打包为 ZIP 导出。

## 使用

1. 打开应用。
2. 点击 `+` 导入图片。
3. 上传 PNG 水印，或输入文字水印。
4. 调整位置和样式。
5. 点击 `导出`，在弹窗中设置导出选项。

## 本地运行

可以直接打开 `index.html`，也可以启动本地服务：

```bash
python -m http.server 8080
```

然后打开：

```text
http://localhost:8080
```

## 主要文件

- `index.html` - 页面结构。
- `styles.css` - 布局、主题和界面样式。
- `js/ui.js` - 图片导入、控件绑定、主题切换和导出弹窗。
- `js/canvas-renderer.js` - 预览和水印渲染。
- `js/export.js` - 导出、元数据、尺寸调整、锐化和 ZIP 打包。

## 隐私

Watermark Studio 在浏览器本地处理图片。导入的图片不会上传。
